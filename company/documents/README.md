# company/documents/ — Certificates and attachments

Upload your company's standing documents here as PDFs.
These get attached to bid submissions automatically.

## What to upload

Rename each file clearly before uploading (no spaces — use hyphens):

| What | Suggested filename |
|---|---|
| Certificate of Incorporation | `certificate-of-incorporation.pdf` |
| GST Registration Certificate | `gst-certificate.pdf` |
| PAN Card (company) | `pan-card.pdf` |
| Audited financials FY2022-23 | `financials-fy2022-23.pdf` |
| Audited financials FY2023-24 | `financials-fy2023-24.pdf` |
| Audited financials FY2024-25 | `financials-fy2024-25.pdf` |
| ISO 27001 Certificate | `iso-27001-certificate.pdf` |
| PCI DSS Certificate | `pci-dss-certificate.pdf` |
| Bank details letter | `bank-details.pdf` |
| MSME / Udyam Certificate | `msme-certificate.pdf` |
| Any other certificate | `[descriptive-name].pdf` |

## Tips

- Always use the latest version of each document
- Ensure certificates are not expired before submitting a bid
- If a document has an expiry date, note it in the filename: `iso-27001-valid-until-mar-2026.pdf`
- Completion certificates for projects go in `company/experience/`, not here
