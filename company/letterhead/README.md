# company/letterhead/ — Your company letterhead

Upload your company letterhead here as a single Word file (`.docx`).
All generated bid documents will be built on top of this template, preserving your
logo, header, footer, and branding.

## How to prepare the letterhead

1. Open your existing company letterhead in Microsoft Word
2. Clear all the body text (keep only the header, footer, and any logos)
3. Leave the body area blank — the tool will fill in the content
4. Save as `.docx` format
5. Upload here (suggested filename: `letterhead.docx`)

## What if I don't have a Word letterhead?

- Export your letterhead from Canva or any design tool as a Word file
- Or ask your design team to provide a blank `.docx` with just the header/footer

## Important

- Only one `.docx` file should be in this folder
- The file must be in `.docx` format (not `.doc` or `.pdf`)
- Make sure the page margins in the letterhead match what you normally use
