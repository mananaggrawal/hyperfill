# company/signature/ — Signature image

Upload the authorised signatory's signature here as an image file (`.png` preferred).
This image is automatically inserted into bid documents wherever a signature is required.

## How to prepare the signature image

**Option A — Scan**
1. Sign on white paper, also add your company rubber stamp if you have one
2. Scan at 300 DPI or photograph clearly
3. Crop tightly to just the signature area
4. Save as `signature.png`

**Option B — Digital signature**
1. Sign on a tablet or phone using any drawing app
2. Export as PNG with transparent background
3. Save as `signature.png`

## Requirements

- Format: `.png` (transparent background is best) or `.jpg`
- Only one image file should be in this folder
- The image will be scaled to approximately 4cm × 2cm in documents

## Privacy

This file is excluded from git. It only exists on your local machine / Google Drive.
