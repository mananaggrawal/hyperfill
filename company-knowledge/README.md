# Company knowledge

Reusable knowledge about **your organisation**, shared by every bid. Ships empty —
fill it per [`../docs/setup.md`](../docs/setup.md).

- **[master-data.md](master-data.md)** — single source of truth for identity, offices,
  signatory, financials. Machine twin: `../toolkit/bidder_profile.py`.
- **[profile/](profile/)** — narrative & reference text (deck, business plan, technical &
  security docs) for writing proposals.
- **[submission-documents/](submission-documents/)** — the actual files you attach to bids.
- **[proposal-library.md](proposal-library.md)** — optional live reference to an external
  proposal folder.

Full guide: [`../docs/knowledge-base.md`](../docs/knowledge-base.md).
